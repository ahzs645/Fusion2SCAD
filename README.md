# Fusion2SCAD

Export Fusion 360 parametric designs to OpenSCAD with BOSL2 support.

## Features

- **Parametric Export**: Preserves Fusion 360 user parameters as OpenSCAD variables
- **BOSL2 Integration**: Generates code using the powerful [BOSL2 library](https://github.com/BelfrySCAD/BOSL2)
- **Shape Recognition**: Automatically detects circles, rectangles, and rounded rectangles for clean output
- **Feature Support**:
  - Extrude (including cut operations)
  - Revolve
  - Fillet (noted for manual BOSL2 rounding parameter application)
  - Chamfer (noted for manual BOSL2 chamfer parameter application)
  - Complex sketch profiles converted to polygons

## Fusion 360 → BOSL2 Mapping

| Fusion 360 | BOSL2/OpenSCAD |
|------------|----------------|
| User Parameters | Variables |
| Rectangle + Extrude | `cuboid()` |
| Circle + Extrude | `cyl()` |
| Rounded Rectangle + Extrude | `cuboid(rounding=...)` |
| Complex Profile + Extrude | `linear_extrude() + polygon()` |
| Profile + Revolve | `rotate_extrude() + polygon()` |
| Fillet | `rounding=` parameter |
| Chamfer | `chamfer=` parameter |
| Join Operation | `union()` |
| Cut Operation | `difference()` |
| Intersect Operation | `intersection()` |

## Installation

### Prerequisites

1. **Fusion 360** installed
2. **OpenSCAD** installed (for viewing exported files)
3. **BOSL2 library** installed in OpenSCAD

### Install BOSL2 in OpenSCAD

```bash
# macOS
cd ~/Documents/OpenSCAD/libraries
git clone https://github.com/BelfrySCAD/BOSL2.git

# Windows
cd %USERPROFILE%\Documents\OpenSCAD\libraries
git clone https://github.com/BelfrySCAD/BOSL2.git

# Linux
cd ~/.local/share/OpenSCAD/libraries
git clone https://github.com/BelfrySCAD/BOSL2.git
```

Or download from: https://github.com/BelfrySCAD/BOSL2/releases

### Install Fusion2SCAD Add-in

1. In Fusion 360, go to **Tools** → **Add-Ins** → **Scripts and Add-Ins**
2. Click the **Add-Ins** tab
3. Click the green **+** button next to "My Add-Ins"
4. Navigate to the `Fusion2SCAD` folder and select it
5. Click **Run** to start the add-in

Alternatively, copy the `Fusion2SCAD` folder to:
- **macOS**: `~/Library/Application Support/Autodesk/Autodesk Fusion 360/API/AddIns/`
- **Windows**: `%APPDATA%\Autodesk\Autodesk Fusion 360\API\AddIns\`

## Usage

1. Open a Fusion 360 design
2. Go to **Tools** → **Add-Ins** → **Scripts and Add-Ins**
3. Select **Fusion2SCAD** and click **Run**
4. Choose a save location for the `.scad` file
5. Open the exported file in OpenSCAD

## Example Output

Given a Fusion 360 model with parameters:
- `width = 50mm`
- `height = 30mm`
- `depth = 20mm`

The exported OpenSCAD file will look like:

```openscad
// Generated by Fusion2SCAD
// https://github.com/BelfrySCAD/BOSL2

include <BOSL2/std.scad>

// ============================================
// Parameters (exported from Fusion 360)
// ============================================

width = 50;
height = 30;
depth = 20;

// ============================================
// Geometry (exported from Fusion 360 features)
// ============================================

cuboid([width, height, depth], anchor=BOTTOM);
```

## Limitations

- **Form tools**: Organic/sculpted shapes cannot be exported parametrically
- **Complex assemblies**: Currently exports the root component only
- **Lofts and sweeps**: Basic support, may require manual adjustment
- **Patterns**: Circular/rectangular patterns are expanded, not parameterized
- **Fillets/Chamfers**: Noted as comments; manual application of BOSL2 rounding needed
